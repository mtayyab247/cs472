1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

The jQuery library provides the jQuery function using which we can select elements using css selectors.
e.g: $("p")
While the jQuery object is the element accessed using the jQuery funciton.
e.g var paragraph = $("#para");
paragraph is a jQuery object.
