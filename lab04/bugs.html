Debugging

The following links may help you in the debugging process:

  • Chrome Dev Tools
  • Validators: W3C HTML, W3C CSS, JSLint JSHint
  • Bonus Lecture Slides - Debugging JavaScript
  • Use console.log() to print debug messages in the Javascript console with Firebug or Chrome Dev Tools like you would with System.out.println() in Java
  • Use alert() to popup debug messages in any browser.
