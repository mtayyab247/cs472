1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 

                //code fragment 1
                $("li").each(function(idx, e) {
                    $(e).css(“color”, “yellow”); 
                });
                //code fragment 2
                $("li").each(function() {
                    $(this).css(“color”, “yellow”);
                });
                
                //code fragment 3
                $("li").each(function(idx) {
                    $(this).css(“color”, “yellow”);
                })

All code fragments are equivalent.
In fragment 1 we have both the index and value as arguments and we are using the value arguments which is the current list element.
In fragment 2 we are not using the index and value arguments instead we are using this to get the current list item element.
In fragment 3 we are getting the index argument which the each function provides but we are not using it.
